# Changelog

## 1.1.0
### ✨ Features
- Added an integrated app launcher inside the Select & Control screen.
- Added Share Logs for quick log sharing.
- Added automatic screen lock after inactivity while using the Touchpad.
- Configurable inactivity timeout (15 seconds to 10 minutes).
- Redesigned the About section with dedicated sub-pages (About Project, Based on, What's New, Logs).
- Added quick links to the GitHub repository and the author's 4PDA profile.

### 🎨 Material Design 3 Polish
- Refined Material Design 3 surface hierarchy (tonal elevation over borders).
- Unified shapes (24dp corners), spacing, and typography.
- Cleaner settings screens with improved visual hierarchy and no unnecessary dividers.
- Updated D-Pad design with modern tonal surfaces.
- Improved touch targets (min 48dp) and accessibility.
- Better consistency across Light, Dark, and AMOLED themes.
- Renamed Display Control to Select & Control for better clarity.

### 🛠 Fixes & Improvements
- Improved navigation and screen organization.
- Improved localization for English, Russian, Ukrainian, and Simplified Chinese.
- Removed hardcoded strings and moved all text to resources.
- Minor bug fixes and stability improvements.

**No functionality changes in the UI Polish section. Visual improvements only.**

## 1.0.0
- Initial official release of XRDesk.
- Full Material Design 3 implementation.
- Advanced Theme System with JSON export.
- Support for multiple external displays.
- Touchpad and Media Controls.
- Multilingual support.
- Implemented auto-dim for touchpad screens.
- Package renaming and foundational structure.
